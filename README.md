# Вычислитель <ins>площади и периметра</ins> геометрических фигур

### На данный момент доступны функции для фигур:
- круг *(circle)*
- квадрат *(square)*
- четырехугольник *(rectangle)*
- треугольник *(triangle)*


## Описание функций:
    
### Круг *(circle)*: **circle.py**

- <ins>Площадь</ins>: **area(r)**
    На вход принимает <ins>длину радиуса</ins> круга *(r)*.
    Вычисляет площадь по формуле **S = πR²**

- <ins>Периметр</ins>: **perimeter(r)**
    На вход принимает <ins>длину радиуса</ins> круга *(r)*.
    Вычисляет периметр по формуле **P = 2πR**


### Квадрат *(square)*: **square.py**

- <ins>Площадь</ins>: **area(a)**
    На вход принимает <ins>длину стороны</ins> квадрата *(a)*.
    Вычисляет площадь по формуле **S = a²**

- <ins>Периметр</ins>: **perimeter(a)**
    На вход принимает <ins>длину стороны</ins> квадрата *(a)*.
    Вычисляет периметр по формуле **P = 4a**


### Четырехугольник *(rectangle)*: **rectangle.py**

- <ins>Площадь</ins>: **area(a, b)**
    На вход принимает <ins>длины смежных сторон</ins> четырехугольника *(a, b)*.
    Вычисляет площадь по формуле **S = a * b**

- <ins>Периметр</ins>: **perimeter(a, b)**
    На вход принимает <ins>длины смежных сторон</ins> четырехугольника *(a, b)*.
    Вычисляет периметр по формуле **P = 2 * (a + b)**


### Треугольник *(triangle)*: **triangle.py**

- <ins>Площадь</ins>: **area(a, h)**
    На вход принимает <ins>длину одной из сторон</ins> треугольника *(a)*, и длину падающей на нёё высоты *(h)*.
    Вычисляет площадь по формуле **S = (ah) / 2**

- <ins>Периметр</ins>: **perimeter(a, b, c)**
    На вход принимает <ins>длины всех сторон</ins> треугольника *(a, b, c)*.
    Вычисляет периметр по формуле **P = a + b + c**
